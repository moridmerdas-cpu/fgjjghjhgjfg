# AMEL SELF55 — سلف‌بات فارسی تلگرام

یک سلف‌بات حرفه‌ای فارسی برای تلگرام — بدون پنل وب، همه‌چیز از داخل خود تلگرام مدیریت می‌شود.

## راه‌اندازی سریع

### ۱. دریافت API تلگرام

از [my.telegram.org/apps](https://my.telegram.org/apps) وارد شوید و `API_ID` و `API_HASH` بگیرید.

### ۲. نصب وابستگی‌ها

```bash
pip install -r requirements.txt
```

### ۳. تنظیم متغیرهای محیطی

```bash
cp .env.example .env
# فایل .env را ویرایش کنید و API_ID و API_HASH را وارد کنید
```

### ۴. اجرا

```bash
python app.py
```

ثبت‌نام و اتصال سلف کاملاً از طریق ربات تلگرام (دستور `/start`) انجام می‌شود؛ نیازی به مرورگر یا آدرس وب نیست.

---

## استقرار روی Render.com

1. ریپازیتوری را در GitHub آپلود کنید
2. در Render.com یک Background Worker جدید بسازید
3. `render.yaml` به‌صورت خودکار تنظیمات را اعمال می‌کند
4. متغیرهای `API_ID` و `API_HASH` را در Environment Variables وارد کنید
5. دیپلوی کنید — کاربران با زدن `/start` روی ربات تلگرام، شماره‌شان را تایید و وارد می‌شوند

---

## دستورات تلگرام

بعد از لاگین، در هر چت تلگرام این دستورات را تایپ کنید:

| دستور | توضیح |
|---|---|
| `سلف روشن` | فعال‌سازی بات |
| `سلف خاموش` | غیرفعال‌سازی |
| `وضعیت` | نمایش وضعیت همه قابلیت‌ها |
| `راهنما` | نمایش همه دستورات |
| `تنظیم دشمن` | ریپلای روی پیام کاربر |
| `منشی روشن` | فعال‌سازی منشی خودکار |
| `ضد حذف روشن` | فعال‌سازی ضد حذف |
| `فونت [0-8]` | تغییر فونت پیام‌ها |

برای لیست کامل: `راهنما`

## ساختار پروژه

```
amel-selfbot/
├── app.py          # اجرای اصلی: event loop مشترک + استارت ربات تلگرام و سلف‌ها
├── telegram_bot.py # ربات اصلی تلگرام (ثبت‌نام، تایید ادمین، منوها)
├── bot.py          # کلاینت تلگرام + هندلرهای سلف
├── database.py     # مدیریت دیتابیس (Supabase)
├── db_cache.py     # کش محلی (SQLite)
├── config.py       # تنظیمات
├── requirements.txt
└── render.yaml     # تنظیمات Render.com (Background Worker)
```

## نکات مهم

- **این یک self-bot است** — فقط روی اکانت خودتان استفاده کنید
- رعایت محدودیت‌های API تلگرام الزامی است
- اسپم را با احتیاط استفاده کنید تا اکانت بن نشود
- Session رمزنگاری شده در دیتابیس ذخیره می‌شود
- ورود کاربران با تایید ادمین و اشتراک‌گذاری شماره از طریق تلگرام انجام می‌شود؛ هیچ رمز عبور پنلی وجود ندارد
